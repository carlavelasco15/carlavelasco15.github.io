PROYECTOS

Pla Comunitari Roquetes - Prescripció social

El sistema permite que los trabajadores del CAP Roquetes (Barcelona) puedan registrar los pacientes a actividades organizadas por entidades del barrio, generar tickets de reserva físicos y hacer un seguimiento de los pacientes para optimizar su tratamiento y mejorar su calidad de vida.

La aplicación permite a las entidades encargadas gestionar sus propias actividades y hacer seguimiento de la asistencia y el desempeño de los pacientes.

Tecnologías: Symfony, MySQL, JavaScript y Twig.

CIFO Sabadell (Vallès) - Larabikes

Proyecto final del curso de Laravel (210h) en el centro CIFO Sabadell (Vallès). Página web para la compra y venta de motos entre particulares. La plataforma incluye un frontend desarrollado en Blade de Laravel, un sistema de autorización y autenticación de usuarios, y una API para la gestión de las motos. Permite a los usuarios subir, gestionar y eliminar motos, así como hacer ofertas a otras motos publicadas.

Tecnologías: Laravel, MySQL, Blade y API Rest.
